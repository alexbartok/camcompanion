# CamCompanion

An independent iOS companion app for viewing and saving footage from a Wi-Fi dashcam.

This repository hosts the app's public pages:

- **Support:** https://alexbartok.github.io/camcompanion/
- **Privacy Policy:** https://alexbartok.github.io/camcompanion/privacy.html
